# Coding Agent Model and Memory Research OS

这个站点是一个长期维护的学习与研究系统，用来支撑一条主线：

> 以模型后训练为主线，以 Coding Agent Memory 为研究问题，以 Coding Agent 为工程落地场景。

它不是普通的学习笔记集合，而是把路线规划、每周执行、每日进度、论文阅读、实验记录和工程项目放进同一个可维护系统里。

## Overview

Overview 专题说明整体要学什么，以及这些内容为什么放在一起。

其中包括：

- 整体路线：目标、主线、阶段和维护原则。
- Agent Training：模型后训练、数据合成、rollout、reward 和 eval。
- Memory：Coding Agent 的 episodic、semantic、procedural memory。
- 科研提升：论文卡片、实验日志、写作和工程组织。

## 周计划

周计划负责把长期路线拆成可执行的一周。

每一周是一个独立目录，目录下包含：

- 本周计划：本周 A 线、B 线、每日安排和交付物。
- 每日进度：当天学习进展、卡点、产出和明日调整。

## 专题

专题用于沉淀较深入的调研内容。

例如 Multica Memory 调研不会塞进周计划正文，而是单独形成专题文档；周计划只链接它并安排每天读什么。

## 项目

项目页用于记录中长期工程方向。

小实验代码可以放在 `labs/`，成熟项目应该单独建 GitHub repo，然后在这里维护项目说明、实验报告和关键结论。

## 模板

模板用于保证长期维护时格式稳定。

当前包含周计划、每日进度、论文笔记、实验记录和周复盘模板。
